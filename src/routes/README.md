# Wilson Sons - Sistema de Agendamento de Visitas Operacionais

Este projeto foi desenvolvido como parte do curso **Desenvolvedor Fullstack JR 2026** (+Prati). O front-end foi idealizado e estruturado com o auxílio da plataforma **Lovable**, permitindo uma prototipagem ágil e uma interface responsiva focada na experiência do usuário.

## 🚀 Sobre o Projeto

O sistema é uma solução para o agendamento e gestão de visitas aos terminais da Wilson Sons. O objetivo é garantir conformidade com as normas de segurança do trabalho (NR-12 e NR-29), educando o visitante sobre procedimentos operacionais através de um vídeo de imersão e um quiz obrigatório.

### 🛠️ Funcionalidades
*   **Agendamento Inteligente**: Seleção de datas e turnos com controle de disponibilidade em tempo real.
*   **Validação de Segurança**: Quiz obrigatório para liberação de acesso.
*   **Integração Back-end**: API em Google Apps Script para persistência de dados em Google Sheets.
*   **Design Responsivo**: Mobile-first, garantindo acessibilidade em diversos dispositivos.

## ⚙️ Tecnologias
*   **Front-end**: React, TypeScript, Tailwind CSS e Material UI (gerado via **Lovable**).
*   **Roteamento**: TanStack Router.
*   **Backend & Banco de Dados**: Google Apps Script e Google Sheets.
*   **Deploy**: Vercel.
